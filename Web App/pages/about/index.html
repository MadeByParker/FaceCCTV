Frequently Asked Questions

What is FaceCCTV?

How does it work?

FaceCCTV uses Deep Learning Artificial Intelligence to recognize facial features in pictures using the Bounding Box technique and then it uses ESRGAN to enhance the detected faces.

Why was it created?

FaceCCTV was created as part of my Final Year Dissertation Project for my Computer Science Degree and I wanted to try and help policing reduce the number of crimes occuring in the UK.

Troubleshooting Guide

I get an error when trying to upload a file.

If you get an error message relating to uploading a file. Then it is likely that:

  • The format of the photo is not supported The photo that is uploaded is in a format that is not supported by the application, so changing photo or exporting the same photo in a accepted format in order to be used. Accepted formats are: JPEG, JPG, PNG.
  • The photo is too big to upload This could mean that the size of the photo is too large, currently FaceCCTV operates on images with a maximum size of 3840*2160 pixels (4K).
